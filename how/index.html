Lightstream
What is it? -  How does it work? -  How do I get it?


01. Preparation

A file is arranged into smaller pieces, a hash of the file is added as meta-data



02. Sending

Color light is emitted from the screen, the receiver processes the light and converts to data



03. Receiving

Each frame is then recombined into a sequence on the receivers device



04. Verifying

The transmission is complete. The hash is checked against the original source to verify
