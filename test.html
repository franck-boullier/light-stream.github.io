Lightstream

Features - Technology - Applications - Questions - About - Download

LightStream is a safe cross-platform transfer technology

Features

- Transfer data with out leaving behind meta-data
- Creates an impenetrable barrier between devices
- Think of Lightstream as a virtual contamination-free usb-stick
- Lightstream can send data to many people in the same room

Technology

- Data is split into smaller packets
- Color light is emitted from the screen, the receiver camera processes the color information and converts it into data
- Each packet is then recombined on the receivers device
- When the transmission is complete. A Hash of the file is checked against the original file to see if the data is complete

Applications

- Android to iPhone transfers in a heartbeat
- Interact with public computers with out tethering to them
- Send the same file locally to 60 people at once
- Super quick way to share clipboard dumps between devices
- Send sensitive documents, like PDF, without touching the internet.
